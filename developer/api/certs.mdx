## 获取控制台 API Token

前往控制台获取：https://console.bitiful.com/apiToken

## 证书内容获取

<Tip>
  最佳实践

  在自有架构中定时运行 **证书内容获取 API** 并结合部署相关脚本，可以做到全局 **自动应用最新的域名证书**。
</Tip>

暂无任何参数

**请求示例**：

```shell
curl -H "Authorization: {API Token from: https://console.bitiful.com/apiToken}" "https://api.bitiful.com/cert/data/<cert_name>/content"
```

**返回示例**：

```json
// curl -H "Authorization: {API-Token}" "https://api.bitiful.com/cert/data/usercontents.somecdn.com-rsa-auto/content"
{
    "crt": "-----BEGIN CERTIFICATE-----\n<证书内容>\n-----END CERTIFICATE-----",
    "key": "-----BEGIN PRIVATE KEY-----\n<私钥内容>\n-----END PRIVATE KEY-----",
    "auto": true //是否自动证书
}
```

<Warning>
  在自动获取证书的工作流中，请注意配合安全措施 **防止 API-Token 泄露**。
</Warning>
