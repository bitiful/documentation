---
title: '存储费用'
description: '对象存储存储容量费用细则。'
---

<Tip>与大多数主流对象存储采用固定费用不同，S4 采用阶梯计费，旨在尽可能降低您的存储成本。</Tip>

| 容量（GB） | 单价 |
| --- | --- |
| `0 ~ 50` | 免费。 |
| `51 ~ 1000 ` | 0.065元/GB/月 |
| `1001 ~ 10000` | 0.055元/GB/月 |
| `10001 ~ 100000` | 0.045元/GB/月 |
| `100000 + ` | 0.04元/GB/月 |

<Info>
**计费细节**

1. 单位为 TB、GB、MB、KB、Bytes，即：1 TB = 1000 GB = 1000000 MB 以此类推；
2. 计费最小单位为 1GB/账户，即：当前账户 A、B、C 三个桶，容量为 10.5 + 20.5 + 30.5 = 61.5GB 共则系统会按 12GB 计费（前 50GB 免费）；
3. 每个小时 S4 产生的出口流量数可抵扣当前小时相应容量的存储费用，即：该小时 A、B、C 三个桶的出口流量总共为 40GB，则该小时存储费用全免（12GB存储费用被40GB流量抵消）
4. **没有** 64KB的单个文件最小计费限制；
5. **没有** 90天 的最低计费时间；

</Info>
